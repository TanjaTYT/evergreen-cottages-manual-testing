# TC-001 Navigation Menu Active State Verification
## Objective

Verify that the active page is visually highlighted in the navigation menu.

## Environment

- Desktop  
- Mobile

## Preconditions

Website homepage is accessible.

## Test Steps
1. Open homepage
2. Observe top navigation menu
3. Navigate to another page
4. Observe navigation menu again
## Expected Result

Current active page should be visually highlighted.

## Test Type

UI Testing

## Status

Failed
