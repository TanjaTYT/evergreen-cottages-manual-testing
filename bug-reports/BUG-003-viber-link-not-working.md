# BUG-003 Viber Link Not Working
## Environment
- Mobile
## Severity

High

## Priority

High

## Preconditions

Website homepage is accessible on mobile device.

## Steps to Reproduce
1. Open homepage on mobile device
2. Scroll to contact section
3. Click Viber link
## Expected Result

Viber application opens or valid Viber contact link is triggered.

## Actual Result

Viber link does not open and no action is triggered.

## Status

Open
