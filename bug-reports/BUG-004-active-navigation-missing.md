# BUG-004 Active Navigation Missing
## Environment
- Desktop
- Mobile
## Severity

Medium

## Priority

Medium

## Preconditions

Website homepage is accessible.

## Steps to Reproduce
1. Open homepage
2. Navigate to different sections/pages
3. Observe top navigation menu
## Expected Result

Current active page should be visually highlighted in the navigation menu.

## Actual Result

No visual indication is shown for the active page.

## Status

Open
