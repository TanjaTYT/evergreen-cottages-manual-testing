# Bug Reports

This folder contains documented defects identified during manual testing of the Evergreen Cottages website.

## Included Defect Types
- Functional defects
- UI defects
- Navigation defects
- Content issues
- External link issues
## Format Used

Each defect is documented in an individual file using:

- Environment
- Severity
- Priority
- Preconditions
- Steps to Reproduce
- Expected Result
- Actual Result
- Status
